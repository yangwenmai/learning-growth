### 发现面试题

同样都是火焰图，CPU 火焰图和内存火焰图，在生成数据时到底有什么不同？
CPU 火焰图和内存火焰图，最大的差别其实就是数据来源的不同，也就是函数堆栈不同，而火焰图的格式还是完全一样的。

- 对 CPU 火焰图来说，采集的数据主要是消耗 CPU 的函数；
- 对内存火焰图来说，采集的数据主要是内存分配、释放、换页等内存管理函数。
